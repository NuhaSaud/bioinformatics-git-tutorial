
# Quick Reference — Command Line

### Command Line Cheat Sheet

| Command | Purpose | Example |
|----------|----------|----------|
| `git status` | Check what’s changed in your working directory | `git status` |
| `git add` | Stage changes for the next commit | `git add protocol.md` |
| `git commit` | Save a snapshot of staged changes | `git commit -m "Add PCR protocol"` |
| `git push` | Upload your commits to GitHub | `git push` |
| `git pull` | Download new commits from GitHub | `git pull` |
| `git branch` | List, create, or delete branches | `git branch` |
| `git checkout` | Switch to another branch or create one | `git checkout -b new-feature` |
| `git merge` | Combine another branch into the current branch | `git merge feature-branch` |
| `git diff` | Show differences between file versions | `git diff` or `git diff --staged` |
| `git log` | View commit history in compact graphical form | `git log --oneline --graph --decorate --all` |
