
# 03 — Collaboration (VSCode)

## Clone
`Ctrl+Shift+P` → **Git: Clone** → SSH URL

## Pull Requests
1. Push feature branch
2. GitHub panel → **Create Pull Request**
3. Request review → address comments → merge

## Conflicts (Visual)
- Open conflicted file → Merge Editor: Accept Current/Incoming/Both → commit

### Exercise
- Two learners edit the same line in `SOP.md` on different branches, open PR, resolve visually.
