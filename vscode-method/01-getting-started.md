
# 01 — Getting Started (VSCode)

- Install **VSCode**
- Extensions: *GitLens*, *Markdown All in One*, *GitHub Pull Requests*

## First Repo
1. `File → Open Folder` → create `lab-protocols`
2. `Ctrl+Shift+P` → **Git: Initialize Repository**
3. Create `README.md` and `protocols/dna-extraction.md`
4. Source Control → stage `+` → commit
5. `Ctrl+Shift+P` → **Git: Add Remote** (SSH URL), then **Publish Branch**
