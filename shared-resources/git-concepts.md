
# Git Concepts & Terminology (Fast Primer)

- **Repository (Repo):** A folder tracked by Git (`.git` holds history)
- **Commit:** A saved snapshot with a clear message (what/why)
- **Branch:** A line of work for experiments/features
- **Remote:** The copy on GitHub for backup/collaboration
- **Clone vs Fork:** Clone = local copy; Fork = your own copy on GitHub
- **Stage → Commit → Push:** Select changes → save locally → upload
