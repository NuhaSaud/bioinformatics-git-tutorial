
# Common Collaboration Workflows

## Feature Branch Flow
- Pull latest → create branch (`add-protocol/dna-extraction-v2`) → commit small changes → push → PR → review → merge

## Docs & Releases
- Tag meaningful states (`v1.0-protocols-ready`)
- Use Markdown for SOPs; keep raw data out of Git

## Roles
- Author → Reviewer → Approver; use PR templates/checklists
